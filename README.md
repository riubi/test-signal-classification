# Кластеризация сигналов сцинтилляционного детектора

Кластеризация осциллограмм со сцинтилляционного детектора на три группы:
гамма-кванты, нейтроны и сигналы, не поддающиеся уверенной классификации.
Решение без учителя на основе разделения по форме импульса (Pulse Shape Discrimination, PSD).

## Данные

`data/Run200_Wave_0_1.txt` - сигналы, разделитель пробел, 504 столбца на строку.
После отбрасывания служебных столбцов остаётся 500 отсчётов осциллограммы на сигнал.
Импульс отрицательный относительно постоянного уровня (baseline ~14820).

## Решение

1. Признаки. Baseline и шум по предымпульсной области; инверсия импульса; заряд методом
   сравнения гейтов. Главный признак - PSD = (q_long - q_short) / q_long, доля заряда в хвосте.
   Дополнительно амплитуда, положение пика, ширина на полувысоте, моменты формы, число пиков.
2. EDA. Распределение PSD бимодально (две полосы). Полосы диагональны: PSD падает с энергией.
   Разделимость FOM растёт с энергией и падает ниже 1 при малом светосборе.
3. Аномалии. Жёсткие правила убирают аппаратный брак (насыщение, наложение, подшумовые,
   испорченный baseline) - менее 0.1%. Область ниже порога FOM=1 относим к кластеру не
   поддающихся классификации сигналов.
4. Разделение частиц. PSD детрендирован по энергии, затем смесь двух гауссиан делит
   разделимую область на гамма (малый хвост) и нейтрон (большой хвост).
5. Сравнение. KMeans, MiniBatchKMeans, GaussianMixture (diag/full/tied), Birch на всём наборе;
   Agglomerative, DBSCAN, Spectral на подвыборке. Внутренние метрики у всех близки, поэтому
   модель выбрана по интерпретируемости и устойчивости.

## Результат

Доля кластера аномалий по данным однозначно не определяется, поэтому подготовлены варианты от
0.1% до 33% и отправлены на Kaggle. Точность растёт при уменьшении доли аномалий:

| вариант | доля аномалий | accuracy |
|---|---|---|
| hard_only (основной) | 0.1% | 0.826 |
| p05 | 5% | 0.804 |
| p12 | 12% | 0.760 |
| fom | 23% | 0.686 |
| p33 | 33% | 0.615 |

Вывод: класс аномалий в разметке это аппаратный брак (около 0.1%), а низкоэнергетическая
область отнесена к частицам. Итоговый ответ submission.csv это вариант hard_only (accuracy 0.826).

Разделение гамма/нейтрон также подобрано по лидерборду: детрендированный PSD с гейтом SW=14
(0.826) точнее, чем SW=8 (0.824) и порог по сырому PSD без детренда (0.708). Формат ответа:
index,cluster, нумерация кластеров по порядку первого появления.
